### PASO 1: Habilitar GitHub Pages
1. **Ve a tu repositorio:** https://github.com/omniversodigital/omniverse-geckos
2. **Settings** → **Pages** (menú lateral izquierdo)
3. **Source:** Deploy from a branch
4. **Branch:** main
5. **Folder:** / (root)
6. **Save**
